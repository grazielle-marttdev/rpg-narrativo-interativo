## 🌌 Sobre o Projeto

Algo integrado ao Cosmo, espalhado por sua imensidão — sem forma definida, sem localização confirmada.

Algumas pessoas sentem. Poucas entendem. E apenas uma foi chamada pelo nome.

**Espaço Consciência** é um conto interativo no estilo *visual novel* sobre o que acontece quando a realidade começa a vibrar de um jeito que não tem explicação.

---

## 🎮 Estilo de Jogo
- Narrativa imersiva com efeito *typewriter*
- Escolhas que ramificam a história
- Música ambiente e efeitos sonoros por cena
- Visual inspirado em *visual novels* com estética cósmica

## 🛠️ Tecnologias
- React + TypeScript + Vite
- CSS puro (ou TailwindCSS no futuro)
